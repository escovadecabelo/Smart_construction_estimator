# Update Log - Smart Construction Estimator

## [Pre-Alpha] - 2026-03-24

### Initial Release
This version establishes the foundation for the automated estimating workflow.

### Added
- **Core Architecture**: Vite-based project structure with a professional dark theme.
- **Branding**: Integrated Mardegan Construction / Restart Builder Services identity and custom SVG logo.
- **PDF Service**: Client-side text extraction service for project bid files.
- **Scope Analyzer**: Heuristic-based engine to detect Sq Ft and work categories (Painting, Demo, Cleaning).
- **Cleaning Specialist**: Expert-level breakdown for post-construction cleaning phases (Rough, Final, Touch-up).
- **Certification Module**: Dynamic management of professional credentials (OSHA, EPA, etc.).
- **Proposal Generator**: High-end PDF/Print template for estimate submission.

### Changed
- Refined Dashboard UI for better file upload visibility.
- Updated company naming logic to distinguish between Painting/Demo and Cleaning sectors.

### Fixed
- Fixed minor CSS compatibility issues with `background-clip`.
- Resolved a syntax error in the PDF upload event listener.
