# Smart Construction Estimator

![Mardegan Logo](/public/logo.svg)

**Smart Construction Estimator** is a specialized tool designed for U.S.-based construction companies (Mardegan Construction & Restart Builder Services) to streamline the bidding process. It automates the extraction of project scopes from PDFs and generates professional, competitive estimates with a focus on Painting, Demolition, and Post-Construction Cleaning.

## 🚀 Version: Pre-Alpha

This is the initial internal release focusing on core automation and specialized cleaning logic.

## ✨ Key Features

- **AI-Powered PDF Analysis**: Upload bid documents from PlanHub, BuildingConnect, or GCs to automatically extract square footage and work scope.
- **Cleaning Specialist Agent**: Detailed breakdown of post-construction cleaning phases:
  - **Rough Clean**: Debris removal and initial sweep.
  - **Final Clean**: Deep deep-cleaning of all surfaces.
  - **Touch-up / Final Shine**: Perfect state for project handover.
- **Certification Management**: Dynamically highlight your credentials (OSHA 30, EPA Lead-Safe, MBE, etc.) on every proposal.
- **Professional Proposals**: Export high-end, branded PDF estimates with your logo, company details, and signature fields.
- **Premium UI**: Modern dark-mode dashboard with micro-animations and smooth transitions.

## 🛠️ Tech Stack

- **Frontend**: Vite + Vanilla JS
- **Styling**: Vanilla CSS (Custom Glassmorphism Design)
- **PDF Engine**: PDF.js (Client-side extraction)

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/escovadecabelo/Smart_construction_estimator.git

# Enter the directory
cd Smart_construction_estimator

# Install dependencies
npm install

# Run the development server
npm run dev
```
